# Data

The shear layer dataset is **embedded** in `realdata/shear_layer.py`.

Source: Wang, L. and Ding, Y. (2009). An integrated method for gear degradation
assessment and remaining useful life prediction. *Technometrics*, 51(4),
doi:10.1198/TECH.2009.07011.

No external data files are required to reproduce the paper results.
